# sfb-editor
Tool to edit PS3_DISC.SFB files, original by pink1stools
```
 * LegacySDK SFB Fork
 * Original SFB by pink1stools, https://github.com/pink1stools/PS3_DISC.SFB-Editor
 * Modified by miskaa (gh:miskkaaa dc:misko.bin)
 * Made for LegacySDK PS3-Launcher
```
```
sfb.exe (PS3_DISC.SFB)
Arguments:

--info / -i | Shows information about the SFB file
--create <output> <TITLE_ID> <TITLE> | Creates a SFB File
```
